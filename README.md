# Neumorphic Tic-Tac-Toe (iOS)

[English] | [日本語] | [中文]

## English
A simple and elegant Tic-Tac-Toe game built with SwiftUI. 
- **Features**: Smart AI algorithm, Neumorphic UI design, Sound effects.

## 日本語
SwiftUIで開発したニューモーフィズム・デザインの三目並べゲームです。
- **特徴**: 独自のAIアルゴリズム（思考ロジック）、直感的なUI、効果音再生機能。
- **技術スタック**: Swift, SwiftUI, AVFoundation.

## 中文
使用 SwiftUI 开发的拟物风格三子棋游戏。
- **核心**: 具备防守意识的智能AI，拟物化视觉设计，实时交互音效。
